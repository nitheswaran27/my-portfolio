# 💼 My Portfolio Website

Welcome to my personal portfolio website! This site is designed to showcase my skills, projects, and experience as a web developer. It’s built entirely using **HTML**, **CSS**, and **JavaScript** – no external frameworks!

🌐 **Live Preview:** [View Website](https://my-portfolio-t4da.vercel.app/)

---

## 🚀 Features

- Fully responsive design (mobile & desktop)
- Smooth scroll navigation
- Project showcase with details
- About me & skills section
- Contact form with EmailJS integration
- Clean and modern UI

---

## 🛠️ Built With

- HTML5
- CSS3 (Custom styling, no frameworks)
- JavaScript (Vanilla)
- EmailJS (for contact form functionality)
